# Farcaster MiniApp

Mini-app sederhana untuk Farcaster dengan tombol **Post GM**.  
Klik tombol untuk simulasi posting “GM” (Good Morning).

## Cara Deploy

1. Buat repo GitHub baru.
2. Paste `package.json` dan `server.js` ke repo.
3. Deploy ke **Vercel**:
   - Login ke [https://vercel.com](https://vercel.com)
   - Pilih **New Project → Import Git Repository**
   - Pilih repo kamu → Deploy
4. Buka URL publik → tombol Post GM muncul
